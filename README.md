# Welcome to my first webpage
## With This project I aim to accomplish many goals:

- ### **First: Familiarize Myself With Git/Github:**
As a first year student, it is highly important that I both incorporate version control into my workflow and build my project portfolio.\
\
Git has only become more important with the huge surge in the use of AI for development, the risks for instability and downtimes has only gotten higher, for this reason the ability to go back to a more stable version is vital.\
\
Github allows me to prove my competencies through my repos, which is important as employers look for individuals who are capable of building and maintaining projects.

- ### **Second: Learn Modern Frameworks:**
As of writing this (21st of March, 2026), this project only contains plain HTML, CSS, and JS, with hopes of incorporating React, Tailwind, and FastAPI:
 - **React** is rich with features, has a large ecosystem and with React Native allows for development of all sorts of apps.
 
 - **Tailwind** allows for cleaner CSS code, its extensive tooling hastens the process of creating appealing styles for your webpage.
 - **FastAPI** is a modern and robust Python framework, it shines in terms of deploying ML models, which is not in the scope of this project, but as I am still exploring my available options, it would benefit me to use this framework.

 The goal is to incorporate all of React, Tailwind and FastAPI by 21st of September of the same year.

- ### **Third: Coding By Hand:**
Though AI is a useful tool for building, relying on it is dangerous for learning, as delegating hands-on work to something else steals your opportunity to learn. In short terms, AI is useful when you already know what it's doing, why it's doing it, and what to check for and how to fix it if something ever goes wrong.\
So I plan on keeping AI usage minimal; for boilerplate and the such, while building features by hand to gain experience and knowledge on how things actually work.
